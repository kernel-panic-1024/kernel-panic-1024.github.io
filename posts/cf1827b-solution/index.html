Codeforces 1827B Range Sorting 题解

发表于 2023-09-22 分类于 题解

思路

考虑如何计算一个长度为 nnn 的序列的代价．

显然可以直接排序，所以答案有上界 n−1n - 1n−1．由于一段区间的代价是 r−lr - lr−l，即区间内形如 i∼i+1i \sim i + 1i∼i+1 的间断数．那么我们需要拆出尽量多的线段进行操作，代价就是必须保留的间断数．

考虑一个间断 i∼i+1i \sim i + 1i∼i+1 不能被舍弃的条件．若 max⁡j≤iaj>min⁡j>iaj\max\limits_{j \le i} a_j > \min\limits_{j > i} a_jj≤imax​aj​>j>imin​aj​，那么 arg max⁡j≤iaj\argmax\limits_{j \le i} a_jj≤iargmax​aj​ 和 arg min⁡j>iaj\argmin\limits_{j > i} a_jj>iargmin​aj​ 必须需要交换，而舍弃这个间断会导致 [1,i][1, i][1,i] 和 [i+1,n][i + 1, n][i+1,n] 中的元素无法交换，所以这个间断必须保留．

考虑对所有子区间计算．由于有上述分析，将贡献拆到间断上，即对每一个间断 i∼i+1i \sim i + 1i∼i+1，计算有多少 [l,r][l, r][l,r]，满足 max⁡l≤j≤iaj>min⁡i<j≤raj\max\limits_{l \le j \le i} a_j > \min\limits_{i < j \le r} a_jl≤j≤imax​aj​>i<j≤rmin​aj​．扫描 iii，由于左边是一段后缀的 max⁡\maxmax，考虑使用单调栈维护，对于单调栈中相邻的两个元素 p,qp, qp,q，开始于区间 [p+1,q][p + 1, q][p+1,q] 内的后缀的 max⁡\maxmax 都是 apa_pap​，我们所要统计的就是有多少个 rrr 满足 ap>min⁡i<j≤raja_p > \min\limits_{i < j \le r} a_jap​>i<j≤rmin​aj​，这个倒着再扫一遍就行．

注意维护单调栈时，弹完栈后，设栈顶为 ppp，那么 l∈[1,p]l \in [1, p]l∈[1,p] 和 r∈[i,n]r \in [i, n]r∈[i,n] 都能产生贡献，记得加上去．
