Codeforces 1861E Non-Intersecting Subpermutations 题解

发表于 2023-09-03 分类于 题解

被数数薄纱了．

不拆贡献，天诛地灭．

思路

首先观察到，对于一个特定的序列，我们可以贪心地去计算划分数，能划分就划分，不会影响答案．

于是考虑设 fi,jf_{i, j}fi,j​ 表示考虑了前 iii 位，目前无重复数字的后缀长度为 jjj 的序列个数．

每个满足要求的子串对答案的贡献为 111，故答案为

∑i=1nfi,k×kn−i\sum_{i = 1}^n f_{i, k} \times k^{n - i} i=1∑n​fi,k​×kn−i

考虑 fi,jf_{i, j}fi,j​ 的转移：

  1. 考虑在长度为 jjj 的后缀后添加一个在该后缀中没有出现过的数字，转移有两种类型
    • 上次未凑出完整的无重复数字的子串：fi,j←(k−j+1)×fi−1,j−1,2≤j≤kf_{i, j} \leftarrow (k - j + 1) \times f_{i - 1, j - 1}, 2 \le j \le kfi,j​←(k−j+1)×fi−1,j−1​,2≤j≤k．
    • 上次凑出来了：fi,1←k×fi−1,kf_{i, 1} \leftarrow k \times f_{i - 1, k}fi,1​←k×fi−1,k​．
  2. 如果添加了一个出现过的数字，由于我们考虑的是所有序列，所以会从后缀长度 ≥j\ge j≥j 的所有状态转移一次：fi,j←fi−1,pf_{i, j} \leftarrow f_{i - 1, p}fi,j​←fi−1,p​，j≤p<kj \le p < kj≤p<k．

显然可以使用前缀和优化．
