洛谷 3732 [HAOI2017] 供给侧改革

发表于 2023-05-27 分类于 题解

来自可爱学妹的带 log⁡\loglog 巨大常数做法．

思路

首先，由于数据随机，不妨猜测 LCP 的最大长度不超过某个小常数 KKK．记 SiS_iSi​ 表示由 iii 开始的后缀的长度为 KKK 的前缀．

询问比较诡异，猜测难以高效地在线解决，于是考虑离线．将询问挂在右端点上，扫描线解决每个询问．对于当前扫描到的右端点 rrr，维护 Xi=data⁡(i,r)X_i = \operatorname{data}(i, r)Xi​=data(i,r)，询问就是对 XiX_iXi​ 区间求和，丢到线段树上即可．

考虑移动右端点，加入 SrS_rSr​ 时对 XiX_iXi​ 的影响．由于 KKK 较小，我们不妨暴力修改．具体而言，我们枚举 SrS_rSr​ 的前缀 TTT，钦定 TTT 为 LCP，然后找到最大的 ppp，使得 SpS_pSp​ 含有 TTT 这个前缀．对于 Xj(1≤j≤p)X_j(1 \le j \le p)Xj​(1≤j≤p)，TTT 作为 LCP 都能对其造成贡献，XjX_jXj​ 需要对 ∣T∣|T|∣T∣ 取 max⁡\maxmax．

如何找到 ppp？考虑 Trie 的性质，对于一个串 SSS，它的一个前缀在 Trie 上对应的节点在 SSS 对应的节点的祖先链上．那么我们维护一棵插入了 Si(1≤i<r)S_i(1 \le i < r)Si​(1≤i<r) 的 Trie，对 Trie 上每个节点 uuu 维护 mxu\mathrm{mx}_umxu​，表示 uuu 对应的串最后一次作为前缀出现是在 SmxuS_{\mathrm{mx}_u}Smxu​​ 中．那么对于 SrS_rSr​ 的一个前缀，设它在 Trie 上对应的节点为 uuu，我们要找的 ppp 就是 mxu\mathrm{mx}_umxu​．

对于区间取 max⁡\maxmax 操作，乍一看需要一棵维护区间历史最值的线段树．然而我们发现，对于 u<vu < vu<v，XuX_uXu​ 中会包含 XvX_vXv​ 的贡献，即 XiX_iXi​ 单调不增，所谓的取 max⁡\maxmax 操作其实是对某个右端点为 ppp 的区间的覆盖，线段树维护区间最小值就可以二分出区间左端点．这样就只需要一棵支持区间覆盖，求区间最小值和区间和的线段树．

在 Trie 上枚举前缀，复杂度省掉一个 KKK．

时间复杂度 O(qKlog⁡n)O(qK\log n)O(qKlogn)，KKK 取 323232 即可通过本题．
