Codeforces 1817D Toy Machine 题解

发表于 2023-09-05 分类于 题解

Ad-hoc．/qd

组题人我谢谢你．

思路

容易 通过瞎摁 观察到，当 k≤⌈(n−2)/2⌉k \le \lceil (n - 2) / 2 \rceilk≤⌈(n−2)/2⌉ 时，可以通过 k−1k - 1k−1 次 LDRU\texttt{LDRU}LDRU 加上一次 L\texttt{L}L 将 kkk 转到最左边．

考虑 k>⌈(n−2)/2⌉k > \lceil (n - 2) / 2 \rceilk>⌈(n−2)/2⌉ 时怎么做，我们可以通过镜像上面的操作将目标点转到右上角，然后重复最多 ⌊n/2⌋\lfloor n / 2 \rfloor⌊n/2⌋ 次 RDRU\texttt{RDRU}RDRU 将所有的块堆到右上角，此时进行 RU\texttt{RU}RU 操作就转化成了 k=⌈(n−2)/2⌉k = \lceil (n - 2) / 2 \rceilk=⌈(n−2)/2⌉ 的情况，直接做即可．

代码

#include <cstdio>
#define debug(...) fprintf(stderr, __VA_ARGS__)

inline int rd() {
	int x = 0, f = 1, c = getchar();
	while (((c - '0') | ('9' - c)) < 0)
		f = c != '-', c = getchar();
	while (((c - '0') | ('9' - c)) > 0)
		x = x * 10 + c - '0', c = getchar();
	return f ? x : -x;
}

int n, k;

int main() {
	n = rd(), k = rd();
	int mid = n / 2;
	if(k <= mid) {
		for(int i = 1; i <= k - 1; i++) printf("LDRU");
		puts("L");
	} else {
		int t = n - k - 1;
		for(int i = 1; i <= t - 1; i++) printf("RDLU");
		printf("R");
		for(int i = 1; i <= mid; i++) printf("RDRU");
		printf("LU");
		for(int i = 1; i <= mid - 1; i++) printf("LDRU");
		puts("L");
	}
	return 0;
}
