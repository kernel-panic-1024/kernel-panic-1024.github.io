Codeforces 1849F XOR Partition 题解

发表于 2023-08-31 分类于 题解

学了两年 OI 跟没学一样．

思路

不妨二分答案．

考虑如何判定存在大于 xxx 的划分方式，我们需要将异或和小于 xxx 的数对划分到两个集合中．我们在这些点之间建边，判定是否是二分图即可．

由于边数是 O(n2)O(n^2)O(n2) 的，直接做无法通过本题．

不妨将原序列排序，设排序之后的序列为 aia_iai​，我们尝试证明一个引理：

若 iii 和 i+4i + 4i+4 之间有连边，那么一定不合法．

若 iii 和 i+4i + 4i+4 之间有连边，那么有 ai⊕ai+4<xa_i \oplus a_{i + 4} < xai​⊕ai+4​<x．

考虑 ai⊕ai+4a_i \oplus a_{i + 4}ai​⊕ai+4​ 的最高位，我们将 ai∼ai+4a_i \sim a_{i + 4}ai​∼ai+4​ 的这一位单独拉出来，显然比这一位高的只会是 000，且这几位的情况只可能是 000010000100001，000110001100011，001110011100111，011110111101111 这 444 种，而前两种中，i∼i+2i \sim i + 2i∼i+2 这三之间两两都可以连边，后两种中，i+2∼i+4i + 2 \sim i + 4i+2∼i+4 这三之间两两也可以连边．

于是我们只需要考虑 iii 与 [i+1,i+3][i + 1, i + 3][i+1,i+3] 这个区间中的点的连边．边数 O(n)O(n)O(n)，总复杂度 O(nlog⁡n)O(n \log n)O(nlogn)．
